# Soil-Moisture-Water-Plants-Notification-by-Blynk

Hey friends in this video I will show you how to make a Soil Moisture sensor Notification system for plants using Blynk

# Connections:- 

NodeMCU --- Soil Sensor
D4 --- D0
GND --- GND
VCC --- 5V

# Materials:- 

1-NodeMCU ESP8266
2-Soil Moisture Sensor
3-BreadBoard
4-Jumpers
